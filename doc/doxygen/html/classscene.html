Detailed Description

One of the main components of the game, contains entities and is played once at a time. A scene typicaly is a level in a game.

To make the game, you must create its entities, with their attributes (see their specific documentation) and then add them to a scene. Once all the entities and scripts are added to the scene, simply run it with the method run()
