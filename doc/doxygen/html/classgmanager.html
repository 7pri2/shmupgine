Detailed Description

This class manages the graphical ressources for the project.

To add pictures to your project, set sprites through graphicrenderers, you must use the gmanager. It is a singleton, so you can't create an object of this type. To add a texture to the project, use `gmanager::new_texture( "texture_name", "path_to_texture", width, height);', then you can use is later for example with the attribute graphicrenderer, you can use the method `set_texture("texture_name");'.
